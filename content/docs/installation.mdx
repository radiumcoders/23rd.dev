## Local development

```bash
pnpm install
pnpm registry:build
pnpm dev
```

Open [http://localhost:3000/docs](http://localhost:3000/docs).

## Add a registry item

1. Create a component in `registry/hello-world/hello-world.tsx`
2. Add an entry to `registry.json`
3. Build the registry:

```bash
pnpm registry:build
```

4. Write docs in `content/docs/components/`

## Registry namespace

Consumers can point `@23rd` at your deployed registry:

```json
{
  "registries": {
    "@23rd": "https://23rd.dev/r/{name}.json"
  }
}
```
