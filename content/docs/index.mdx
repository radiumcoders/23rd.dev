**23rd** is a component registry with documentation powered by Fumadocs MDX and a custom floating layout — no default Fumadocs UI styles.

## What you get

- MDX docs with a floating sidebar
- A shadcn-compatible registry under `registry/`
- Built JSON items served from `/r/[name].json`

## Quick start

1. Add components under `registry/`
2. Register them in `registry.json`
3. Run `pnpm registry:build`
4. Document them in `content/docs/`

Install a published item:

```bash
pnpm dlx shadcn@latest add https://23rd.dev/r/hello-world.json
```

Or from this repo once deployed, add the namespace:

```bash
pnpm dlx shadcn@latest registry add @23rd=https://23rd.dev/r/{name}.json
pnpm dlx shadcn@latest add @23rd/hello-world
```
